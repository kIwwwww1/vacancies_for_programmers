# ITJobsBot

Бот для поиска работы и размещения резюме для программистов. Позволяет удобно публиковать вакансии и резюме прямо в Telegram

## Основные возможности
- Размещение резюме для поиска работы.
- Публикация вакансий для программистов.
- Просмотр всех доступных вакансий и резюме.
- Удобный интерфейс в Telegram.

## Как использовать

1. Найти бота в Telegram: [IT_Jobs_Officially_Bot](https://t.me/IT_Jobs_Officially_Bot) или **ITJobsBot**.  
2. Отправить команду для размещения резюме или вакансии.  
3. Просматривать публикации других пользователей в канале: [DevJobsHub](https://t.me/DevJobsHub) или **IT Вакансии & Резюме**.  
4. Быстро находить подходящие вакансии или кандидатов.


## Основные технологии
- **Python**
- **SQLAlchemy** – работа с базой данных.
- **aiogram** – асинхронный фреймворк для Telegram.
- **asyncio** – асинхронное выполнение задач.
- **Docker** – контейнеризация приложения.
- **Docker Compose** – удобное управление сервисами.
